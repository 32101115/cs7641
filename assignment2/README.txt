

To run this program, 
1. cd into ABAGAIL folder
2. Type ant to compile 
3. Type 
java -cp ABAGAIL.jar opt.test.BreastCancerTest [population size] [number of mate] [number of mutate]

java -cp ABAGAIL.jar opt.test.FpFlopTest [first parameter] [second parameter] [third parameter]

java -cp ABAGAIL.jar opt.test.TravelingSalesmanTest [first parameter] [second parameter] [third parameter]

java -cp ABAGAIL.jar opt.test.KnapsackTest

for example:
java -cp ABAGAIL.jar opt.test.BreastCancerTest 500 200 50
